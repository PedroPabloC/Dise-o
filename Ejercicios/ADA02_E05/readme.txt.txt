1.- Para compilar cualquier aplicación, primero tienes que dirigirte a la línea de comandos y ubicarte en alguna carpeta "problema#".
2.- Después compilar la clase main de alguno de los problemas. Ejemplo: nombreDelPrograma.java (Escribir "javac DoubleLinkedTest.java" o "MoviesApp.java" o main del problema 1)
3.- Ejecutar cualquier programa. Puedes hacerlo con el siguiente comando "java nombreDelMain.java"(Por ejemplo, "java MoviesApp.java")
4.- Siempre se debe ejecutar el programa correspondiente a la carpeta y al nombre del main
Adicional: También se puede ejecutar en cualquier editor de texto aplicando los pasos correspondientes a cada uno, en el caso de VSC solo hay que darle a "RUN" o "Run program".
	   Para cambiar los valores y ejecutar cualquier método de algún programa, hay que hacerlo modificando el main.
